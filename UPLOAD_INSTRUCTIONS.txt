RECOMMENDED WEB-INTERFACE PROCEDURE

2. Create a backup branch from main named:
   backup/pre-final-profile-site
3. Return to main.
4. Select Add file > Upload files.
5. Drag README.md and the complete docs folder from this extracted package.
6. Commit with this message:
   Deploy final David Barske profile website
7. Open Settings > Pages and confirm:
   Deploy from a branch / main / docs
8. Wait for deployment, then open the public route and press Ctrl+F5.
